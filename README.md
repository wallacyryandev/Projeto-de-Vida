📚 TheLegacy: Ecossistema de Evolução Pessoal
"A tecnologia abre a porta. O inglês te deixa entrar." — A filosofia central deste projeto.

O TheLegacy é um projeto de front-end desenvolvido por mim para fins de estudo pessoal e gestão de metas. Ele não é apenas um site, mas um dashboard interativo que mapeia minha jornada de transição de carreira: do concurso público à imigração para o Canadá como desenvolvedor de IA.

🎯 O Objetivo do Projeto
Este sistema foi criado para centralizar e monitorar três pilares fundamentais da minha rotina de estudos:

Estabilidade: Foco no concurso da PM-PE.

Tecnologia: Aprendizado de Python, IA e Automação.

Internacionalização: Preparação para o IELTS e imigração canadense.

🏗️ Estrutura do Software
O projeto é composto por três interfaces principais integradas:

Dashboard Principal (index.html): Apresenta o roadmap das 3 fases da missão, curadoria de canais educativos e o plano estratégico para o Canadá.

Gestor de Estudos (estudos.html): Um sistema de checklist interativo que calcula o progresso em tempo real nas matérias de Português, Matemática, Direito e Python.

Showcase de Projetos (projetos.html): Uma galeria que documenta o status de desenvolvimento dos meus projetos reais (concluídos, em andamento e planejados).

🛠️ Tecnologias e Conceitos Estudados
Desenvolvi este projeto aplicando os seguintes conceitos de desenvolvimento web:

UI/UX Moderno: Uso de variáveis CSS para temas, glassmorphism, animações de scroll e design responsivo para dispositivos móveis.

Manipulação de DOM: Lógica em JavaScript puro para atualizar barras de progresso e contadores dinamicamente.

Persistência Local (LocalStorage): O progresso dos estudos é salvo no navegador do usuário, permitindo que os dados persistam sem a necessidade de um backend complexo nesta fase.

Intersection Observer API: Utilizada para disparar animações apenas quando os elementos entram na visão do usuário.

🚀 Como utilizar
Como este é um projeto de estudo próprio focado em tecnologias estáticas:

Clone o repositório.

Abra o index.html em seu navegador.

O progresso marcado no Dashboard de Estudos será mantido no seu navegador através do localStorage.
